# U.S. Colleges Data Visualization Dashboard

## Overview
This project visualizes and analyzes U.S. college data to help compare institutions based on cost, enrollment, admission rates, graduation outcomes, and geography.

## Tools
- Python (Pandas, NumPy)
- Visualization: Matplotlib, Seaborn, Plotly
- Environment: Anaconda, Jupyter Notebook
- Dataset: College Scorecard (Kaggle)

## Visualizations
- Tuition comparison by college type
- Enrollment distribution
- Tuition vs graduation rate
- Interactive geographic map of U.S. colleges

pip install -r requirements.txt
